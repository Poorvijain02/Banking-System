# Banking System using Java

A console based Banking Management System developed in Java demonstrating Object Oriented Programming principles and Role Based Access Control.

## Features

* Admin Login
* User Login
* Account Creation
* Balance Inquiry
* Deposit Money
* Withdraw Money
* Transaction Logging
* Role-Based Access Control (RBAC)

## Technologies Used

* Java
* OOP Concepts
* ArrayList
* Scanner Class

## Project Structure

* Account Management
* Authentication System
* Transaction Management
* Activity Logging

## Future Enhancements

* Database Integration (MySQL)
* GUI using Java Swing
* Password Encryption
* File-based Data Persistence

## Learning Outcomes

This project helped in understanding:

* Classes and Objects
* Encapsulation
* Data Structures
* Authentication Systems
* Banking Application Workflow
